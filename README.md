# 🐾 Petshop System

Sistema de gerenciamento de Petshop desenvolvido em Java, com integração ao banco de dados MySQL utilizando JDBC.

---

## 🚀 Funcionalidades

- Cadastro de clientes
- Cadastro de animais
- Agendamento de consultas
- Listagem de dados
- Integração com banco de dados MySQL

---

## 🛠️ Tecnologias utilizadas

- Java 17+
- JDBC
- MySQL
- Swing (interface gráfica)
- Maven
